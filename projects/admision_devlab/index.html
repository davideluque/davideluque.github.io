Pregunta #1 ¿Qué es una hoja de estilos o stylesheet?

Las hojas de estilos son un conjunto de instrucciones las cuales se ocupan de los aspectos de formato y presentación de los contenidos.

Pregunta #4 ¿Cuál es la importancia y la utilidad de las clases y los id de las etiquetas de HTML al momento de personalizar dichos elementos mediante el uso de hojas de estilo CSS?

Las clases son útiles para aplicar estilo a un cierto grupo de elementos sin tener que repetir el código por cada elemento. También permiten hacer una distinción entre el formato y estilo de una página, y entre el diseño como tal. Los ids a pesar de que no es buena práctica aplicarle un mismo identificador a dos elementos en una misma página ya que se usan para identificar un elemento único en la página, permiten aplicar el estilo a dicho elemento separando el código del estilo del elemento como tal.

CSS

Pregunta #1 ¿Qué son archivos CSS u hojas de estilo? ¿Por qué consideras que son útiles al momento de diseñar una página web?¿Por qué se prefiere el uso de hojas de estilo antes que el uso de estilos inline?.

CSS se refiere a Cascading Style Sheets y describe cómo los elementos HTML se muestran en la pantalla. Puede controlar el diseño de múltiples páginas web a la vez. Las hojas CSS u hojas de estilo son archivos que dan formato a una página web mediante un conjunto de reglas que son aplicadas a los elementos de dicha página. Las hojas de estilos son vitales ya que permiten que una página web tenga una presentación más vistosa mediante la organización y aplicación de estilos de tales elementos. El uso de hojas de estilo sobre el css in-line es preferido principalmente porque permite reutilizar la misma hoja de estilos para varias partes de una misma página, por otra parte hace que el código HTML se vea más limpio y ordenado y así el trabajo queda más organizado.

Pregunta #6 Mencione y explique brevemente algunas de las propiedades CSS que consideres más importantes.

background ← permite añadir un fondo a un elemento ya sea mediante un fondo, un degradado o un archivo externo
color ← aplica un color a una fuente ya sea en hexadecimal, rgb, o con un nombre soportado por css.

Javascript

Pregunta #1 ¿Qué hace el método document.getElementById de Javascript?

Selecciona el elemento que haga match con el ID y devuelve su objeto en el DOM. Similar al selector $(‘#id’) de jQuery.

Pregunta #4 ¿Qué hacen los métodos alert() y console.log?

alert(‘mensaje’) muestra en forma de cuadro de diálogo emergente en el navegador un mensaje específico. El usuario debe presionar OK para quitar el mensaje.
console.log() sirve para mostrar en la consola del modo de desarrollador o debugging lo que se pase por parametro a esté metodo, ejemplo console.log(var) imprimiría en la consola el valor de var.

Pregunta #5 ¿Qué son los eventos del DOM de HTML? Mencione algunos de estos eventos y explique brevemente qué hace o significa cada uno.

Los eventos del DOM permiten a los lenguajes que soportan la programación orientada a eventos como Javascript, ECMAScript registrar varios manejadores de eventos en los nodos del DOM. Por ejemplo, se puede registrar el evento “click” en un elemento

<p> de manera tal que cuando un usuario navegue por la página y haga clic en el elemento p, se desencadene una acción o función que podría manipular el DOM si se desea. Algunos eventos son:
onfocus ← cuando un usuario selecciona un campo
onclick ← cuando un usuario hace click con el mouse en un elemento
onsubmit ← cuando se envía un formulario
onchange ← cuando el elemento sufre algún cambio
